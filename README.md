# 🚀 Cloud Delivery Pipeline Portafolio

![CI Pipeline](https://github.com/vermaldonado-ia/cloud-delivery-pipeline-portafolio/actions/workflows/ci.yml/badge.svg)

Repositorio que demuestra la implementación de un **pipeline DevOps completo**, incorporando:

- Integración Continua (CI)
- Validación de calidad de código (Quality Gate)
- Control de Pull Request (PR)
- Despliegue Continuo simulado (CD)

---

## 🎯 Objetivo

Construir un pipeline que permita:

- Detectar errores de forma temprana  
- Asegurar calidad del código  
- Validar cobertura de pruebas  
- Controlar merges a rama principal  
- Simular un flujo de despliegue real
